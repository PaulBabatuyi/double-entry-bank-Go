# Go Double-Entry Ledger

Minimal backend demonstrating double-entry accounting principles in Go for a simple bank ledger.

Features (planned):
- Atomic deposits, withdrawals, transfers with opposing debit/credit entries
- Immutable ledger (entries table as source of truth)
- PostgreSQL + sqlc + Gin

Tech: Go, PostgreSQL, golang-migrate, sqlc, Gin, Docker
